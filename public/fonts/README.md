# Fonts

Drop the Scriptina font file here, named **scriptina** with one of these
extensions (any one is enough): `.woff2` (best), `.woff`, `.ttf`, `.otf`.

Example: `public/fonts/scriptina.woff2`

The @font-face in src/styles/global.css already references these paths, so the
script titles ("Augenblicke", "Erleben", …) will use it automatically.
Tip: convert .ttf/.otf to .woff2 at https://transfonter.org for smaller size.
